#### python decorator 
decorator let you add extra behaviour to a function , without chnaging the function code 
a decorator is function that takes another function as input and return function 

def changecase(func):
    def myinner():
        return func().upper()
    return myinner
@changecase
def myfunction();
    return "Hello sally"
 
print(myfunction())
